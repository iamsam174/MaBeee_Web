# MaBeee_Web

**MaBeee_Web App**は、乾電池型IoTデバイス「MaBeee（マビー）」をBluetooth経由で手軽にコントロールするWebアプリです。

PC・タブレット端末・スマートフォンからURL `https://iamsam174.github.io/MaBeee_Web/` にアクセスするだけですぐに利用できます。

このアプリはWeb上からBluetoothを操作する技術（Web Bluetooth API）を使用していて、現在 Google Chrome と Microsoft Edgeでのみサポートされています。 

Safari や Firefox では動作しませんので、ご注意ください。

⚠️ Appleのセキュリティ方針により、iPad (iPadOS) / iPhone (iOS)のSafari や Chrome ではBluetooth通信機能（Web Bluetooth）がブロックされている関係で、このアプリ単独では動きません。

⚠️ iPad / iPhone で動かしたい場合は「Bluefy `https://apps.apple.com/jp/app/bluefy-web-ble-browser/id1492822055`」や「WebBLE `https://apps.apple.com/jp/app/webble/id1193531073` 」といったWeb Bluetooth専用のブラウザアプリでお使いください。

---

## ✨ 主な機能

- **多様な動作モード:** 利用者の障害特性や操作の習熟度に合わせて、以下の4つの動作モードを切り替えられます。
  1. **クリックまたはタップでタイマー実行モード:** マウスポインタをボタン内に移動させて左クリックする、または、ボタンをタップすると、タイマー設定した時間（1秒〜180秒）MaBeeeがONになります。
  2. **マウスオーバー（マウスポインタがボタンに触れた瞬間）でタイマー実行モード:** マウスポインタがボタンに触れた瞬間にタイマーが起動し、一定時間MaBeeeがONになります。
  3. **マウスポインタがボタン内にある間マビーONモード:** マウスポインタがボタンの領域内にある間だけMaBeeeがONになり続け、マウスポインタがボタンの外に出るとOFFになります。
- **Web Bluetooth API連携:** 面倒な専用ソフトのインストールは不要です。ブラウザの機能だけで直接MaBeeeとBluetoothペアリングを行います。
- **UIカスタマイズ:** 対象物の見やすさに合わせて、操作ボタンのサイズ（特大、大、中、小）や操作音（電子音、ファンファーレなど）が変更可能です。

## ⚠️ 動作環境（必須条件）

ブラウザのセキュリティ制限（Web Bluetooth APIおよびカメラアクセス）のため、以下の環境が必要です。

- **対応ブラウザ:** Google Chrome または Microsoft Edge（※SafariやFirefoxでは動作しません）
- **対応OS:** Windows 10/11, macOS, Android OS, Chrome OS
  - *(※iPad / iPhoneなどのiOS/iPadOS環境では、標準ブラウザSafariがWeb Bluetoothをブロックしているため動作しません。「Bluefy」や「WebBLE」といったWeb Bluetooth専用のブラウザアプリが必要です)*
- **ハードウェア:** 
  - Bluetooth機能（BLE）を搭載したICT機器（WindowsPC、Mac、Android端末、Chromebook）
  - 単４乾電池を入れたMaBeee（※緑色のMaBeee for Schoolとは通信仕様が異なります）および乾電池で作動するオモチャなど
  - USB外付けWebカメラ（乾電池式オモチャなどの対象物をボタンに映し出したい時に使用します。スマートフォンやタブレット端末など、機種によっては外付けWebカメラの映像をボタン背景に映し出す機能が利用できないことがあります。その場合は内蔵カメラを仕様するかPC環境での利用をオススメします。）

## ⚙️ ペアリングと操作手順

1. 画面上の **［MaBeeeに接続する］** ボタンをクリックします。
2. ブラウザ上部にBluetoothデバイスのリストが表示されるので、「MaBeee...」から始まるデバイスを選択して「ペア設定」をクリックします。接続が途切れる時は、ブラウザをリロードしてみてください。また、MaBeeeと電池ボックスの接点に隙間がないかを確認したり、MaBeeeに入れた単４乾電池を新品に交換してみてください。
3. **［外付けWebカメラ］** を使う場合、それぞれのボタンを押して、ブラウザからのカメラ使用許可を「許可」にします。
4. 動作モードを設定します。ラジオボタンをクリックして設定するだけでなく、キーボードの［ `1` ］キー［ `2` ］キー［ `3` ］キーでも設定できます。
5. 動作モード①を選択してタイマー秒数を設定します。スライダーをマウスドラッグして動かして設定するだけでなく、キーボードの［ `←` ］キーまたは［ `→` ］キーでも設定できます。
6. 利用者の実態に応じてボタンサイズを切り替えます。ボタンサイズは特大・大・中・小の４つがあり、ドロップダウンメニューから選べます。
7. アプリ上のアクセス内容に応じて音（「ピコーンをという音、または、ファンファーレ）鳴らすことができます。音なしも含めてドロップダウンメニューから選べます。
8. 画面右下の **［🚀 スタート（実行画面へ）］** ボタンを押して実行画面へ移行します。
9. 実行画面に **［🤖 マビーON］** ボタンが表示されますので、それにアクセスするとMaBeeeを入れた乾電池式オモチャなどが動きます。
10. 起動画面がブラウザからはみ出る場合は、ブラウザのズーム機能で縮小してみてください。
11. MaBeeeとの接続を終わらせるには、使用している端末のブラウザで開いているページをリロードするかタブを削除します。その後、スマートフォン・タブレット端末・Chromebookは、アプリスイッチャーでブラウザを終了させてください。WindowsPCやMacはブラウザを終了すればMaBeeeとの接続が切断されます。
